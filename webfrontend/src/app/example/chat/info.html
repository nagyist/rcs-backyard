General info

This is a GUI to demonstrate how to use sails.js and WebSockets to automatic update between individual clients. This GUI shows you an example about chat feature which you can easily do with angular and sails.js. Basically this is "the best" demonstrative for live update features.

Backend

  • Message.js (model)
  • MessageController.js (controller)
  • controller.js (base controller)

Frontend

  • chat.js (module configuration)
  • chat-component.js (chat component, contains controller, directive and service)
  • chat.html (template)
  • KeyPress.js (key press directive)
