General info

This GUI demonstrates how you can use $sailsSocket service to fetch data from backend for your GUI and how you can use ui.bootstrap.pagination component for that data. Also this will demonstrate how you can sort that data by simply click table header.

This GUI has following basic actions attached to it.

  • Data fetch from server via WebSocket communications
  • Data pagination
  • Data sort by column

Every time when user changes page or sort order data is been fetched from server via $sailsSocket. Used codes for this GUI is listed below.

Backend

  • Book.js (model)
  • BookController.js (controller)
  • controller.js (base controller)
  • Book.json (initial data)

Frontend

  • books.js (module configuration)
  • books-controller.js (controller)
  • books.html (template)
  • DataService.js (data service)
