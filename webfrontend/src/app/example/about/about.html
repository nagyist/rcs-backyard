Sails.js + AngularJS Boilerplate

This is an example boilerplate project for your Sails.js + AngularJS combo. Basically this project contains following components:

  • Backend (Sails.js)
    • Only API, nothing frontend stuff
    • Passport.js for user authentication (local, twitter, facebook, g+, etc.)
    • JWT (JSON Web Tokens) support for API calls
    • CORS support
  • Frontend (AngularJS)
    • Lightweight only client side application
    • Authentication to backend
    • Route examples for views
    • Support for WebSocket communication with backend
    • Interceptor for HTTP / Socket communications
